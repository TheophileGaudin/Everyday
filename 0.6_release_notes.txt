**New widget**

- Implemented a subtitles widget for phone output. The subtitle engine runs on the phone. English only.
  - Loads Sherpa ONNX on first start.
  - Toggle it manually, allow on phone.
  - Will subtitle phone output.

**New features and improvements**

1. Registered the app on Google Cloud for verification.
2. Added layout save/load.
3. Reworked the context menu so multiple options are easier to navigate in the limited real estate of the FOV.
   - Tap on a subcontext menu to reveal options.
   - Scroll when there are many options.
   - Double tap to move to the parent context menu when in a subcontext menu.
4. Double tap on the border areas of widgets now reveals the context menu of the app rather than activating the double tap functionality of that widget.
5. Browser widget address bar improvements.
   - More intuitive to edit.
   - If not a web address, search Google.
   - Clipboard from phone can be pasted in, for example a YouTube video link.
6. Added alignment guides while dragging/resizing, togglable with the layout menu.

**Bug fixes and maintenance**

1. Fixed inhomogeneous resize behavior.
2. Full screen widget no longer displays a border when hovered outside of the border area.
3. Refactored/unified data sync from phone to glasses.
   - Flow: if phone is connected, refreshes from phone and then sends to glasses; else, if glasses are connected to the internet, they call the API; else, cached data.
