Everyday

Everyday is a lightweight, customizable HUD and widget dashboard for RayNeo X3 Pro AR glasses, with a phone companion app for input, screen mirroring, location data, and Google Calendar authorization.

About Everyday

Everyday is built for glanceable AR use. It lets users place, resize, minimize, and arrange widgets in their field of view so the glasses can stay useful without obstructing daily life.

Glasses App

Runs the HUD experience on RayNeo X3 Pro, including widgets for time, weather, browser, text, images, news, finance, speed, screen mirror, and Google Calendar.

Phone Companion

Provides trackpad and keyboard input, clipboard sync, location data, screen mirroring support, and the Google authentication flow used by the Calendar widget.

Google Calendar

Google sign-in is optional and is used only when a user enables the Calendar widget. Everyday requests calendar read-only access to show upcoming events in the HUD.

Google OAuth Use

Everyday uses Google OAuth so users can choose to connect Google Calendar. The app requests basic account identity and read-only calendar access. It does not sell user data, serve ads, or operate a server that stores Google user data.

  • Account identity is used to show which Google account is connected.
  • Google Calendar read-only access is used to fetch upcoming calendar events for the Calendar widget.
  • Users can disconnect Google from inside the app, which clears locally stored authorization state.
